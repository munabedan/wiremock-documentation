# Dev

The repo includes a few convinience scripts to help setup and run the project in the 'dev' directory.

- Setup

```bash
./dev/setup_env.sh
```

- Serve

```bash
./dev/serve.sh
```

- Build

```bash
./dev/build.sh
```
